Proyecto Mini-API

Pequeño proyecto enfocado a practicar y reflejar mis aprendizajes de Python.
El objetivo es aplicar conceptos fundamentales como la lógica, estructuras de datos, funciones, sintaxis, buenas prácticas, manejo de errores y organización.

Tecnologías utilizadas -> Python 3.10+

Requisitos previos -> Tener instalado Python 3.10 o superior.

Instalación
git clone https://github.com/LMellaVillanueva/mini_api.git
cd mini_api

Ejecución
python mini_api.py

Funcionalidades
- Funciones simples y específicas utilizando List Comprehension.
- Lógica básica que escalará en un futuro.
- Código sencillo y organizado para buenas prácticas.
